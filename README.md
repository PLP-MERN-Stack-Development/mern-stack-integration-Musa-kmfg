Socket.io Chat (Full Version)
============================
This archive contains:
- server/ : Express + Socket.io + MongoDB
- client/ : React + Vite + Socket.io-client (Tailwind-ready)

Quick start:
1. Configure server/.env from .env.example and run `npm install` then `npm run dev`
2. In client: run `npm install` then `npm run dev` (set VITE_API_URL if needed)
